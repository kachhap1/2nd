HTML Formatting Elements

Formatting Elements Description Examples
1 <b> This is a physical tag, which is used to bold the text written between it. This text is bold
2 <strong> This is a logical tag, which tells the browser that the text is important. This text is important!
3 <i> This is a physical tag which is used to make text italic. This text is italic
4 <em> This is a logical tag which is used to display content in italic. This text is emphasized
5 <mark> This tag is used to highlight text. Do not forget to buy milk today.
6 <u> This tag is used to underline text written between it. Write Your First Paragraph in underlined text.
7 <tt> This tag is used to appear a text in teletype. Hello Write Your First Paragraph in monospaced font.
8 <strike> This tag is used to draw a strikethrough on a section of text. Write Your First Paragraph with strikethrough.
9 <sup> It displays the content slightly above the normal line. This is superscripted text.
10 <sub> It displays the content slightly below the normal line. This is subscripted text
